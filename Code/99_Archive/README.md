# 99_Archive

`99_Archive` 用于存放“当前默认不参与编译或启动，但未来仍可能复用，因此不应直接删除”的资产。

## 当前规则

1. 归档内容默认不加入 `CMakeLists.txt`
2. 归档内容默认不参与当前版本主流程
3. 需要恢复使用时，再按需移回主干层或显式加入构建

## 当前已归档内容

1. `01_Task/Inc/task_test.h`
2. `01_Task/Src/task_test.c`
3. `docs/07_毕业设计作品说明.md`

## 归档原则

适合进入 `99_Archive` 的内容通常同时满足以下条件：

1. 当前版本默认不用
2. 未来复用概率高
3. 保留在主干会增加理解成本或构建边界混乱
